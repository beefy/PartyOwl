Party Owl

Down in the dumps? Need a party companion? A fan of owls? Fear no more, for The Party Owl now gives a premium party owl experience to iOS, Android, and Windows phones! With speed action, rotation technology and background color dispersion, Party Owl will party with you using the latest in partying technology. For hours of partying fun, download Party Owl for free, now!

This is a small app for Windows, iOS, and Android. It was developed with Apache Cordova and used Phoneblocks in publishing. The app itself is only two simple modes, and the only difference is a changing background color and a spinning image. To learn about setting up a simple multiplatform Cordova app, or to contribute to this project, you can clone the project as follows:

$ cd your_repo_root/repo_name
$ git clone https://github.com/nschultz14/PartyOwl.git
